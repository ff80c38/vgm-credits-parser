# Changelog

## [1.0] - 2026-02-15

### Added
- Initial public release under GPL-3 license
- Modular parser for common credit formats
- VGMdb integration with Cloudflare workaround
- Interactive text editor with real-time parsing preview
- Automatic vorbis tag mapping for artist roles
- Tag normalization and customization options
- Automatic `ARTIST` and `ALBUMARTIST` calculation
- Export to `.tsv`, `Mp3tag`, and `foobar2000` formats
- Integrated FLAC file tagger (metaflac integration)
